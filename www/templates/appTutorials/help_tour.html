Welcome to the Main Help!

At any given time you might need some help.

Now we are going to explain you how to access for help.

Icons used in help section

  • This icon will allow you to access for help in the specific area of the CPS Model you are currently located in.
  • This icon displays more information.
  • This icon represents a website provided by Lives in the Balance.

Accessing help in a specific help area of the CPS Model:

  1. Tap on the icon.
  2. Search for the topic you need help in.
  3. If a topic has a tap on it and it will display more information.
  4. If this icon appear it means that it is a website provided by Lives in the Balance.

Accessing the Main Help:

  1. Swipe to the left to see the menu options.
  2. Tap on the "Main Help" option
  3. Choose the area of the CPS model you need help with.
  4. Tap on it
  5. Look at the topics we can help you with
  6. Tap on the one you need help with

Using the search bar

  1. Swipe to the left to see the menu options.
  2. Tap on the "Main Help" option
  3. This bar will help you search for a specific topic
