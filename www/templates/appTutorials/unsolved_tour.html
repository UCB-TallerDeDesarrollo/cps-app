Welcome to the Unsolved Problems!

Once you have selected a Lagging Skill you can create an Unsolved Problem.

Now we are going to show you how you can do it.

Icons used in this Step

  • This icon allows you to add a new Unsolved Problem to the current list.
  • This icon allows you to edit the current Unsolved Problem.
  • This icon allows you to delete the current Unsolved Problem.
  • This icon allows you prioritize the Unsolved Problems.
  • This icon allows you to save the changes of priorization you've made on the Unsolved Problems.

Add an Unsolved Problem

Just three steps:
  1. Tap on the icon
  2. Fill the input.
  3. Tap on create button.

Edit an Unsolved Problem

Just four steps:
  1. Swipe the Unsolved Problem to the left to see the options.
  2. Tap on the icon
  3. Edit the Unsolved Problem.
  4. Tap on the Save button.

Delete an Unsolved Problem

Just three steps:
  1. Swipe the Unsolved Problem to the left to see the options.
  2. Tap on the icon
  3. Tap OK in order to confirm the action.

Prioritize the Unsolved Problems

Just three steps:
  1. Tap on icon to enable the drag & drop option.
  2. Tap over icon and drag up or down an Unsolved Problem to prioritize it.
  3. Tap on icon to save the changes.

Start the Plan B

Just one step:
  1. Tap on one Unsolved Problem.
  2. Or Swipe to the left to see options.
  3. Tap on the icon
  4. And tap the First Step: Empathy Step.

Any questions?

You can click the help icon.
